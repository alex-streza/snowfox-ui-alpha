# Row

Rows are layout elements that can contain a group of other elements using flex properties. They can be used to create a grid of elements.

The direction inside a row can be different for each breakpoint: `xs`, `s`, `m`, `l`, `xl` and `xxl` and the supported directions are: `row`, `column`, `row-reverse` and `column-reverse`.
